Instalando no Mac OS X

CouchDBX

O caminho mais fácil para instalar o CouchDB no Mac OS X é baixando o CouchDBX. Está aplicação não oficial não instala qualquer coisa em seu sistema e pode ser executada apenas com um duplo clique. Note, entretanto que para um uso mais sério, é recomendado você fazer a instalação tradicional com alguma coisa do tipo Homebrew.

Homebrew

Homebrew é uma adição recente para as ferramentas de gerenciamento de software do Mas OS X. A premisa é configuração zero, otimizações peasadas e um tema para cerveja. Baixe o Homebrew a partir de http://github.com/mxcl/homebrew. As instruções de instalação são minimas. Uma vez configurado, execute: 

brew install couchdb

no terminal e aguarde até finalizar. Para iniciar o CouchDB, simplesmente execute: 

couchdb

para ver todas as opções de inicialização disponíveis, execute: 

couchdb -h

Este comando mostra como rodar o CouchDB em background, entre outras formas.

Para verificar que o CouchDB está realmente rodando, abra seu browser na url http://127.0.0.1:5984/_utils/index.html.

MacPorts

MacPorts é de fato a ferramenta para gerênciamento de pacote para Mac OS X. Enquanto não é parte oficial do sistema operacional, ele pode ser usado para simplificar o processo ed instalação de software FLOSS na sua máquina. Antes de você poder instalar o CouchDB com MacPorts, você precisa baixar e instalar o MacPorts.

Tenha certeza que sua instalação do MacPorts está atualizada rodando: 

sudo port selfupdate

Você pode instalar o CouchDB com MacPorts rodando: 

sudo port install couchdb

This command will install all of the necessary dependencies for CouchDB. If a dependency was already installed, MacPorts will not take care of upgrading the dependency to the newest version. To make sure that all of the dependencies are up-to-date, you should also run: 

sudo port upgrade couchdb

Mac OS X tem um framework de gerênciamento de serviço chamado launchd que pode ser usado para iniciar, parar e ou gerenciar os daemons do sistema. Você pode usá-lo para iniciar o CouchDB automaticamente quando o sistema iniciar. Se você quiser adicionar o CouchDB para a configuração do launchd, rode: 

sudo launchctl load -w /opt/local/Library/LaunchDaemons/org.apache.couchdb.plist

Após rodar este comando, CouchDB deve estar disponível em:

http://127.0.0.1:5984/_utils/index.html

CouchDB também será inicilizado e parado juntamente com o sistema operacional.
