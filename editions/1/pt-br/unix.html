Instalando em Sistemas Unix

Debian GNU/Linux

Você pode instalar o CouchDB executando: 

sudo aptitude install couchdb

Quando isto estiver completo, você deve ter uma copia do CouchDB rodando em sua máquina. Certifique-se de ler a documentação específica do sistema Debian que pode ser encontrada em /usr/share/couchdb.

Começando com Ubuntu 9.10 (“Karmic”), o CouchDB vem pré instalado com todo sistema desktop.

Ubuntu

Você pode instalar o CouchDB executando: 

sudo aptitude install couchdb

Quando isto estiver completo, você deve ter uma copia do CouchDB rodando em sua máquina. Certifique-se de ler a documentação específica do sistema Ubuntu que pode ser encontrada em /usr/share/couchdb.

Gentoo Linux

habilite o ebuild de desenvolvimento do CouchDB executando: 

sudo echo dev-db/couchdb >> /etc/portage/package.keywords

Verifique o ebuild CouchDB executando: 

emerge -pv couchdb

Monte e instale o ebuild CouchDB executando: 

sudo emerge couchdb

Quando isto estiver completo, você deve ter uma copia do CouchDB rodando em sua máquina.

Problemas

Olhe oApêndice D, Instalando a partir do Código se a sua distribuição não tiver um pacote do CouchDB.
